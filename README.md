# cloudace_linebot

## HOW TO USE
```
$ git clone $(This repository's URL)
$ cd seq2seq/
$ mv ~/Downloads/facebook-$(USER) ./raw/facebook
$ mv ~/Downloads/\[LINE\]\ Chat\ with\ *.txt ./raw/line/
```
